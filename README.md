🚀 Crypto Tracker App

A modern and fully responsive cryptocurrency tracking application built with React. This project allows users to monitor real-time crypto market data, analyze price changes through interactive charts, and explore detailed information about each coin.

📦 Installation

To run this project locally:

# Clone the repository
git clone https://github.com/your-username/crypto-tracker.git

# Navigate into the project directory
cd crypto-tracker

# Install dependencies
npm install

# Start the development server
npm run dev

🔥 How It Works
The app fetches real-time cryptocurrency data using API requests via Axios.
React Context API is used for global state management.
Users can search for any coin and instantly view results.
Clicking on a coin navigates to a detailed page with advanced analytics and charts.
🌍 API

This project uses a public cryptocurrency API to fetch real-time data.
